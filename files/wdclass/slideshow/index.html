How to make a CSS slideshow

A short tutorial by Lea Verou

Structure

Every slide is an article element, with a class of "slide" except the first one which is a header element and the last one which is a footer element.

In more complex slideshows, we could have multiple sections, with their own headers and footers.

Basic idea

  • Every slide has a width and height of 100%, so that it occupies the whole viewport.
  • They also have box-sizing: border-box; to avoid paddings and borders being added to that 100%, which would cause scrollbars to show and make it hard to use.
  • All slides are hidden except the current one

How to reference the current slide?

The current slide can be selected with the :target pseudo-class.

However, there are other acceptable solutions too, like:

  • hiding the body scrollbar with overflow:hidden; (the browser takes care of scrolling to the right place)
  • Positioning the non-current slides off-screen, either with transform: translateX(100%), left: 100%; or variations of these ideas.

Printing

  • You need an @media print { ... } block, with print specific styles.
  • To print 4 slides per page, we have to give them a width and height of at most 50%
  • If we want spacing between them, we have to adjust the width and height to something smaller

Font size

  • If we don’t use JavaScript, we can’t do much about that. There will always be slides with too little or too much content
  • We can only make sure the base font-size is reasonable, and reasonable differs per screen size
  • Media queries to the rescue!
